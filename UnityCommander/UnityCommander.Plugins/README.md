# 📦 Описание файлов и классов библиотеки `PluginLibrary`

## Основные компоненты

- **`PluginLibrary.sln`** — решение для сборки библиотеки, которое включает проект самой библиотеки и проект с тестами.
- **`PluginLibrary.csproj`** — основной проект библиотеки с зависимостями и настройками компиляции.

## Интерфейсы

- **`IPlugin.cs`** — интерфейс для плагинов, который все плагины будут реализовывать. Содержит базовые методы для взаимодействия, запуска и завершения работы плагина.
- **`IPluginContext.cs`** — интерфейс для контекста плагина. Используется для передачи зависимостей, настроек и взаимодействия с хост-приложением.
- **`IPluginFactory.cs`** — интерфейс для фабрики плагинов, которая создает экземпляры плагинов с учетом специфики реализации.

## Реализации и логика

- **`PluginContext.cs`** — реализация контекста плагина. Хранит параметры, сервисы и другие зависимости, необходимые конкретному плагину.
- **`PluginFactory.cs`** — реализация фабрики плагинов. Используется для создания и инициализации плагинов в едином стиле.
- **`PluginMetadata.cs`** — хранит метаданные плагинов: имя, версия, автор, описание, путь к сборке и т. д.
- **`PluginLoader.cs`** — логика загрузки плагинов. Может реализовывать загрузку из DLL, каталогов, а также поддерживать горячую перезагрузку.
- **`PluginContainer.cs`** — класс-хранилище всех активных плагинов, их зависимостей, ресурсов и контекстов. Централизованное место для доступа к плагинам.
- **`PluginManager.cs`** — менеджер жизненного цикла плагинов: загрузка, выгрузка, перезагрузка, очистка ресурсов. Служит связующим звеном между системой и контейнером.

## Тесты

- **`PluginLibrary.Tests.csproj`** — проект юнит-тестов.
- **`PluginContainerTests.cs`** — тесты проверки корректности хранения и доступа к плагинам.
- **`PluginManagerTests.cs`** — тесты управления жизненным циклом плагинов.
- **`PluginContextTests.cs`** — тесты передачи и использования зависимостей.
- **`PluginFactoryTests.cs`** — проверка правильности инициализации и создания плагинов.

## Дополнительно

- **`README.md`** — документация по использованию библиотеки. Содержит инструкции по созданию плагинов, интеграции, загрузке, тестированию и расширению.
