# 🤝 Как внести вклад в UnityCommander

Привет и добро пожаловать! 🎉

Хочешь помочь проекту? Вот как это сделать:

## 🚀 Быстрый старт

1. Сделай форк репозитория
2. Создай свою ветку: `git checkout -b feature/MyFeature`
3. Добавь изменения и сделай коммит: `git commit -m "Добавлена новая фича"`
4. Сделай push: `git push origin feature/MyFeature`
5. Оформи Pull Request

## 📦 Стиль кода

- Используем C# 10 / .NET 9
- Следуем принципам SOLID
- Обязательно пишем комментарии к сложным блокам и публичным методам
- Все новые классы покрываются **юнит-тестами**
