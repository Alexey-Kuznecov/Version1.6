# 📁 UnityCommander

> **Современный файловый менеджер с гибким интерфейсом, плагинами и ИИ для автоматизации работы с файлами.**

---

## 🚀 Описание проекта

**UnityCommander** — это мощный файловый менеджер, вдохновлённый Total Commander и Visual Studio Code, но переосмысленный для современного пользователя. Проект разрабатывается на **C# (.NET 9, WPF, MVVM)** с использованием библиотеки **Prism**, и включает в себя:

- Поддержку **многооконного интерфейса (до 4 панелей)**
- Гибкую систему **плагинов** с автопоиском и изоляцией
- Встроенный **текстовый редактор, просмотрщик изображений и мини-браузер**
- Интеграцию **ИИ-модулей** для автоматической категоризации и очистки файлов
- Лёгкий, настраиваемый **интерфейс с темами и расширениями**
- Высокую **производительность и стабильность**, даже при работе с большим количеством файлов

---

## 🔧 Техническая архитектура

- **Язык:** C#
- **Платформа:** .NET 9.0 (Windows)
- **UI:** WPF + Prism + MVVM
- **Тесты:** xUnit + Moq (80%+ покрытие)
- **Загрузка плагинов:** `PluginLoader`, `PluginContainer`, `ExtendedPluginContainer`
- **Обработка ошибок:** собственные исключения (`InvalidPluginException`, и др.)
- **Искусственный интеллект (в планах):**
  - Категоризация файлов
  - Предсказание часто используемых файлов
  - Очистка мусора и дубликатов
